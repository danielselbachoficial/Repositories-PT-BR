# Registro do Windows

Este subrepositório contém documentação e referências técnicas relacionadas ao **Windows Registry**.

## Conteúdo

- Chaves de registro relevantes para:
  - Segurança
  - Hardening
  - Auditoria
  - Troubleshooting
- Impactos conhecidos de cada configuração
- Diferença entre comportamento padrão e customizado

## Diretrizes

- Nenhuma alteração deve ser aplicada sem entendimento do impacto.
- Sempre realizar backup do registro antes de mudanças.
- Preferir políticas (GPO) quando aplicável.
- Validar comportamento em ambiente de teste.

## Público-alvo

- Administradores de sistemas
- Infraestrutura
- Segurança da informação
